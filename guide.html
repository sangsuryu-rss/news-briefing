1관리 주소
📢
공지 올리기·내리기
…/edit/main/notice.txt
🔄
즉시 뉴스 갱신
…/actions/workflows/update.yml
📁
파일 올리기
…/upload/main
📰
뉴스 사이트
…/news-briefing
2공지사항 올리기 · 내리기 (가장 자주)
1
공지 올리기 주소 접속 → 연필(✏️) 편집 화면.
2
빈 칸에 공지 타이핑 (엔터로 여러 줄).
3
Commit changes → 창에서 다시 Commit changes.
4
1~2분 뒤 Ctrl+Shift+R → 배너 오른쪽에 공지 ✅
▸ 긴급 공지는 맨 앞에 긴급 또는 ! → 빨간색 강조.   ▸ 내리기: 글자 전부 지우고 Commit.
3지금 당장 최신 뉴스로
1
즉시 갱신 주소 → Run workflow → 초록 Run workflow.
2
1~2분 뒤 강력새로고침(Ctrl+Shift+R).
4꼭 기억할 규칙
🚫 로컬 파일 금지
내 컴퓨터 파일 더블클릭 시 공지·날씨 안 뜸. 항상 github.io 주소로.
📤 업로드해야 반영
파일 변경은 GitHub에 Commit 해야 사이트에 적용.
⏱️ 반영엔 1~2분
직후 안 보여도 실패 아님. Ctrl+Shift+R.
